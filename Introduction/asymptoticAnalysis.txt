Asymptotic Analysis is nothing but the theoritical analysis of algorithm without actually writing the algorithm instead with some kind of Pseudo Code.
⁕ The speed of the machine and the programming language is not considered in Asymptotic Analysis.
⁕ When it comes to analysis of algorithms, we always talk about the Input Size and Time of Function that is "Zero or Greater than Zero", because the Size of Input and Function Time can never be Zero.
⁕ In Asymptotic Analysis, Order of growth is measured based on the Input size. 
⁕ If the input size n, the order of growth of function and time taken by the function based on the size of input.
⁕ In Asymptotic Analysis we always talk about the value of n being "Infinity(∞)."

Direct way to find and compare the growth rates:
1. Ignore the Lower Order Terms.
2. Ignore Leading Term Constant.

Examples : f(n) = 2n^2 + 3n + 4   -> Growth Rate : n^2 (Quadratic)
           g(n) = 100n + 3        -> Growth Rate : n (Linear)
